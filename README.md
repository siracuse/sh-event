# Plateforme événementielle sur la Tech

Il s'agit d'un site web où un utilisateur peut poster un événement et participer à d’autres événements.
Ce site inclut un **back office** pour administrer les annonces et gérer les utilisateurs.

## 🎯 Objectifs du projet
L'objectif de ce projet personnel était de m'entraîner sur Symfony 7, et de couvrir l'ensemble des étapes de réalisation d'un site web : le maquettage (wireframe sur papier), l'intégration et le développement du back-end avec le MCD.

## 🌐 Démo
🔗 Site en ligne : [sh-event.sh-dev.fr/](https://sh-event.sh-dev.fr/)

## 🧩 Fonctionnalités principales
- Ajouter / modifier / supprimer des événements  
- Participer à un événement  
- Back office pour administration des annonces et utilisateurs 

## 🛠 Stack
- Symfony 7.4
- HTML/CSS
